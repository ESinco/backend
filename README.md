# ProjetIn

Para configurar o seu ambiente de trabalho rode os seguintes comandos:

## 1. Criar o ambiente virtual
```
python -m venv .venv
```
## 2. Ativar o ambiente virtual
### No Windows
```
.venv\Scripts\activate
```
### No Unix ou MacOS
```
source .venv/bin/activate
```
